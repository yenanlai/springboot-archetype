# SpringBoot项目骨架

基础:
- 统一响应结果
- 统一异常处理
- 返回结果语言国际化
- Swagger
- Feign
- Eureka
- Guava Loading Cache缓存
- Mybatis通用Mapper
- Mybatis Generator自动生成代码插件
- PageHelper分页插件
- HTTP Request日志打印
- Date工具类

测试:
- h2数据库

拓展:
- Redis
- Kafka

提示:
- mv _gitignore .gitignore
- 重构package name
