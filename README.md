# SpringBoot项目骨架

基础:
- 统一响应结果
- 统一异常处理
- 返回结果语言国际化
- Swagger
- Feign
- Eureka
- Mybatis通用Mapper
- PageHelper分页插件
- LoadingCache缓存
- HTTP Request日志打印
- Date工具类

测试:
- h2数据库

拓展:
- Redis
- Kafka
